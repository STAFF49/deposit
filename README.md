# 儲蓄小幫手

高息馬拉松活期存款計算器（純 JavaScript 離線版）。

## 使用方式

雙擊開啟 **`index.html`** 即可，無需安裝 Node 或 npm。

同目錄需有：
- `offers-data.js` — 歷史優惠期資料
- `memo-pixar-gold.png` — MEMO 獎賞圖片（可選）

## 功能

- **利息計算**：三階段利率、每日複利明細、實際年利率
- **歷史優惠期**：展開查看日期、美元利率、MEMO 特別獎賞
- **特別獎賞 MEMO**：符合門檻時顯示提示
- **存款紀錄**：存於瀏覽器 localStorage，支援備註／贈品回贈、三階段明細
- **匯出／匯入 JSON**：備份紀錄至 `save/` 資料夾

## 資料夾

| 路徑 | 用途 |
|------|------|
| `data/offers.json` | 優惠期主資料（編輯用） |
| `offers-data.js` | 離線 HTML 載入用 |
| `save/` | 手動匯出的存款紀錄 JSON |

## 更新優惠資料

1. 編輯 `data/offers.json`
2. 執行 `npm run sync-offers` 產生 `offers-data.js`
3. 重新整理瀏覽器

## 使用說明

1. 輸入本金；存款開始日期預設為**今日**
2. 依日期自動配對優惠期，或從「歷史優惠期」手動選擇
3. 按「計算」查看結果
4. 按「儲存紀錄」保存至存款紀錄頁
